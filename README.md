# LaunchpadXGameController
Using your LaunchpadX as a Game Controller!  

Currently keymapped for No Man's Sky.

Installation:
Requires Python 3 and the mido and keyboard libraries.

> python -m pip install keyboard
> 
> python -m pip install mido[ports-rtmidi]

Then simply run **python lpx_nms.py** from a command window, and minimize it, then run Steam or your game.
